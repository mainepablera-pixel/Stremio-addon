# TMDb Metadata Addon for Stremio

This addon fetches movie metadata from TMDb and displays it in Stremio. Built for mobile use on Huawei NOVA Y73. It enhances posters, ratings, and descriptions for a better viewing experience.

## Setup
1. Replace `YOUR_TMDB_API_KEY` in `addon.js` with your actual TMDb key.
2. Run the addon locally or deploy it to a server.
3. Add the addon to Stremio via its manifest URL.
